# python_study

* 입력
  * 기본 입력 받기
  ```
  l = input()
  ```

  * 입력 받아서 정수로 바꾸기
  ```
  l = int(input())
  ```

  * 입력 받아서 쪼개기
  ```
  l = input().split()
  ```

  * 입력 받아서 쪼갠후 정수로 바꾸기
  ```
  l = list(map(int, input().split()))
  ```

* 루프
  * 0에서 10까지 출력 (for)
  ```
  for i in range(0, 11) :
    print(i)
  ```

  * 0에서 10까지 출력 (while)
  ```
  i = 0
  while i > 10 :
    print(i)
    i++
  ```
